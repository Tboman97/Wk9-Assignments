Class Reflection

In class on 10/28 I learned that the library has more resources than just books. For example, they have a whole online catalogue that I was unaware of. Something interesting that you can do with that catalogue is look-up what books the CCSU library has as well as what books other Connecticut public libraries have. I saw an example of this today when we looked up a famous author and CCSU did not have the book we were looking for, but Southern Connecticut State University's library did.
Libraries support student research in the internet age by giving students a huge database full of credible sources to work from. When students are doing research they can look to that database for websites, books, newspapers, videos, and more that can help them. I think it is interesting that the library's database can be used for more than just phsyical books and I think it greatly helps student research.

Computational Thinking

4 Aspects of Computational Thinking

Decomposition
Breaking down a big problem into the smaller problems that make it up.
Pattern Recognition
Finding the pattern that relates all the different kinds of small problems.
Pattern Generalisation and Abstraction
Giving a general reason for the big problem based off the pattern.
Algorithm Design
Write down the steps it will take to solve the big problem.
